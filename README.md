## Project Overview

### This project was created using:

- Engine: Unity
- Version: 6000.4.0f1
- Template: 3D (Built-in Render Pipeline)

### To open and run this project, you need:

- Unity Hub
- Unity version 6000.4.0f1

⚠️ Opening the project in a different Unity version may cause errors or require upgrading.

### How to Open the Project
- Open Unity Hub
- Go to the Projects tab
- Click Add (or Open)
- Select the root folder of this project (the folder containing Assets, Packages, and ProjectSettings)
- Make sure Unity Hub uses version 6000.4.0f1
- Click Open
